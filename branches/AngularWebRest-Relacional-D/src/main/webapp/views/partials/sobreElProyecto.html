Sobre el Proyecto

Trabajo practico realizado en el Segundo cuatrimestre 2016 para la materia Algoritmos III, UNSAM.
Practica orientada a el desarrollo de una web, utilizando el modelo SPA. Maqueteado y diseñado con html5, CSS3, diversos componentes de CSS como ser Boostrap 3 y creación más adaptación propia. Desarrollado mediante lenguajes de programación como ser javascript(cliente) y xtend (lado servidor). Con ayuda de una simulación de servidor gracias a Jetty y Xtrest.

Point of interest

Tiene como finalidad mediante el uso de servicios REST, el comunicarse con el servidor que contiene diversos puntos de interes para que el usuario pueda averiguar mediante un buscador información relevante y poder dar su opinión respecto a cada punto que desee, compartiendo una puntuación que se generara por el conjunto de usuarios.
Tambien se podra asignas como favoritos lo que permite al usuario tener el recordatorio de que puntos fueron de su interes, y pudiendo modificar este como su comentario.
En la tabla de busquedas se podra visualizar si respecto a su posicion se encuentra cerca o no cada punto, como tambien si se encuentra disponible en este momento por si necesita acudir a ese lugar.
Dentro de los detalles especificos se podra ver a que distancia exacta se encuentra y en la mayoria de los casos que horarios dispone.
